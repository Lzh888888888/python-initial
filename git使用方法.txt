主要用於暫存或恢復用

建立印藏資料夾：
git init

加入暫存區：(進入暫存區檔案旁會顯示「A」)
git add 檔案名(可用「*.py」為所有的python檔案，「.」為全部檔案)

建立還原點：
git commit -m "想用的還原名稱"


檢查當前目錄中的狀態：
git status

檢視還原點歷史(日誌)：
git log
(按下「q」可結束檢視)

簡化的日誌：
git log --oneline

比較新舊版本的差異：
git diff (日誌上要還原的編號 ex.2d73044) -- 檔案名

還原先前的快照：
git checkout (要還原的編號) -- 檔案名
還原到更早的快照(還原後後面的快照會消失)：
git reset  --hard (要還原的編號)

要忽略儲存的檔案：
    先建立「.gitignore」在裡面輸入要排除的檔案(.ex. apple.png或*.png)
